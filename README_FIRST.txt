PEDIATRIC SURVEILLANCE LOOKUP — SIMPLE WEB DEPLOY INSTRUCTIONS

What this folder is:
- A ready-to-upload React website package for Vercel.
- The app data is already embedded in src/App.jsx.

What you do:
1) Create a GitHub account if needed.
2) In GitHub, create a NEW repository.
3) Upload ALL files from this folder into that repository.
4) Go to Vercel.com and sign in with GitHub.
5) Click Add New -> Project.
6) Choose the GitHub repository you just created.
7) Click Deploy.
8) Wait for the site to build.
9) Vercel will give you a live web link.

Important:
- Upload the CONTENTS of this folder, not the zip inside another folder level.
- If GitHub asks, commit the uploaded files.
- If Vercel asks for framework, choose Vite.
- Build command: npm run build
- Output directory: dist

If you want a custom domain later:
- In Vercel open the project
- Settings -> Domains
- Add your domain and follow the prompts

If something fails:
- Take a screenshot of the error and send it back here.
